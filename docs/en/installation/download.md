# Downloading

The panel is distributed through GitHub releases — no source code is published anywhere, only ready-to-run builds.

## Option 1 — through the bot's button

In [@divinepanel_bot](https://t.me/divinepanel_bot)'s main menu, tap **Download**. The button leads to the repository page at [github.com/extremoffff/Divine-Panel](https://github.com/extremoffff/Divine-Panel).

## Option 2 — directly

Open [github.com/extremoffff/Divine-Panel](https://github.com/extremoffff/Divine-Panel). The repository page has a **Download** badge button right under the title — it downloads the right archive immediately.

**[SCREENSHOT PLACEHOLDER: repository page with the Download button]**

## What to actually download

Download the **`DivinePanel-Setup-win64.zip`** archive — that's the only file you need for installation. It already contains both the panel itself and the loader program (Launcher), which takes care of updates for you going forward.

The [Releases](https://github.com/extremoffff/Divine-Panel/releases) page may show another archive with a version number in its name (e.g. `DivinePanel-v1.0.0-win64.zip`) — that one is internal, used by the loader for auto-updates, and you don't need to download it manually.

## What's next

Extract the downloaded `DivinePanel-Setup-win64.zip` into any folder and move on to [First launch](first-launch.md).
