# Activating your license

The token you got from the Telegram bot (see [Cabinet → Tokens](../subscription/cabinet.md)) needs to be entered into the panel.

## Where to enter the token

1. In the panel's sidebar, click the settings icon (the gear) — a popover opens with two items: **Token** and **Notifications**.
2. Choose **Token**.

**[SCREENSHOT PLACEHOLDER: settings popover with Token/Notifications items]**

A window opens with a field for the token and a license status indicator.

**[SCREENSHOT PLACEHOLDER: Token window with input field and status]**

3. Paste the token you copied from the bot into the field.
4. Click **CHECK & SAVE**.

## Status values

Next to the input field, a status pill shows:

- **NOT VERIFIED** — the token hasn't been checked yet.
- **CHECKING…** — a check is in progress.
- **ACTIVE — SAVED (PRO)** or **ACTIVE — SAVED (ULTIMATE)** — the token was accepted, showing your plan. The plan label is always shown in English regardless of the panel's interface language.
- An error message (for example, that the token is already linked to another computer, is empty, or has expired) if something went wrong.

The Token section is available without a subscription (on the FREE plan) — it's exactly what you need to activate one in the first place.

## Linking to your computer

On successful activation, the token links to this computer's hardware (HWID). A single token can't be used on multiple computers at once — trying to activate the same token on a different machine requires resetting the link through the bot first (see [Reset HWID](../subscription/cabinet.md#reset-hwid)).

## What's next

With your token activated, you can start adding Steam accounts — see [Account Manager](../accounts/add-single.md).
