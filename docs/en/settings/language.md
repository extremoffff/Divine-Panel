# Interface language

The panel fully supports Russian and English — switched independently of the subscription Telegram bot's language.

The **Language** button sits at the bottom of the sidebar, below the list of sections.

**[SCREENSHOT PLACEHOLDER: Language button at the bottom of the sidebar]**

Click it — a language selection window opens, with the current choice marked.

**[SCREENSHOT PLACEHOLDER: language selection window]**

Pick the language you want — the panel saves your choice and fully reloads with the new interface language. The choice persists across restarts too.
