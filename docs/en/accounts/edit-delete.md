# Editing and deleting

## Editing

Click the pencil icon (Edit) on an account card — the same form used to add an account opens, already filled in with the account's current data (including its loaded maFile, saved password, PayPass, and comment).

Change whatever needs changing and click **Save**. The required fields are the same as when adding an account: Name, Proxy, Steam username, Steam password, maFile.

## Comment

Separate from the full edit form, each account card has a **+ add comment** button — it opens a small window with a note field. It saves instantly and independently of the rest of the profile's data.

## Trade URL

The ⇄ button on an account card fetches that account's current trade URL from Steam fresh, every time, and copies it to your clipboard — there's no need to keep a static link stored in the panel.

## Deleting

Click the trash icon on an account card. The panel asks for confirmation — deleting an account can't be undone.

**[SCREENSHOT PLACEHOLDER: delete-account confirmation window]**

Once confirmed, the profile is fully removed from the panel along with all its settings (marketplace toggles, thresholds, keys). The Steam account itself and the API keys on the marketplaces aren't deleted — the panel simply stops working with them.
