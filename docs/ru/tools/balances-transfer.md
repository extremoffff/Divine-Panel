# Балансы и переводы

Кнопки **Balances**, **Money Transfer** и **PayPass** находятся на верхней панели страницы **Account Manager** — они работают сразу со всеми добавленными аккаунтами.

## Balances

Показывает баланс каждого подготовленного аккаунта на каждой площадке (market.csgo, Waxpeer, DMarket), а также баланс Steam-кошелька — единым списком по всем аккаунтам. Данные поддерживаются свежими фоновым опросом, отдельно обновлять вручную не обязательно.

## Money Transfer

Позволяет собрать балансы с нескольких аккаунтов в один — например, если нужно объединить небольшие остатки с разных аккаунтов на одном перед выводом.

<figure><img src="../.gitbook/assets/image (54).png" alt=""><figcaption></figcaption></figure>

Этой же функцией — но не из панели, а из личного Telegram-бота уведомлений (не путать с ботом подписки @divinepanel\_bot) — можно перевести весь баланс с одного аккаунта на другой или собрать балансы со всех аккаунтов сразу командой `/transfer`. Подробнее — см. [Личный Telegram-бот уведомлений](../settings/notifications-bot.md#команда-transfer).

## PayPass

Устанавливает платёжный пароль market.csgo сразу на нескольких выбранных аккаунтах за одно действие — не нужно открывать форму редактирования каждого аккаунта по отдельности.
