# История продаж

Раздел **Sales history** (в сайдбаре) — журнал всех прошедших продаж по всем аккаунтам и площадкам. Доступен на тарифах PRO и ULTIMATE.

<figure><img src="../.gitbook/assets/image (52).png" alt=""><figcaption></figcaption></figure>

## Фильтры

Над таблицей — панель фильтров:

* **All accounts** — фильтр по конкретному аккаунту.
* **All markets** — фильтр по конкретной площадке.
* **From** / **To** — диапазон дат.
* Поле поиска по названию предмета.

Кнопка **↻ Refresh** обновляет список вручную.

## Сводка и таблица

Над списком показывается краткая сводка по отфильтрованным продажам. Сама таблица содержит колонки: иконка предмета, **Item** (название), **Account**, **Market** (площадка), **Price**, **Time** (время продажи).

Список постраничный — переключение страниц находится под таблицей.
