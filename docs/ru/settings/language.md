# Язык интерфейса

Панель полностью поддерживает русский и английский языки — переключаются независимо от языка Telegram-бота подписки.

Кнопка **Language** находится внизу сайдбара, под списком разделов.

**[МЕСТО ДЛЯ СКРИНШОТА: кнопка Language внизу сайдбара]**

Нажмите её — откроется окно выбора языка с отметкой текущего выбранного варианта.

**[МЕСТО ДЛЯ СКРИНШОТА: окно выбора языка]**

Выберите нужный язык — панель сохранит выбор и полностью перезагрузится с новым языком интерфейса. Выбор сохраняется и после перезапуска панели.
