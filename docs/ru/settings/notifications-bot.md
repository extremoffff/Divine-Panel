# Личный Telegram-бот уведомлений

Это отдельная функция от бота подписки [@divinepanel\_bot](https://t.me/divinepanel_bot) — панель может отправлять уведомления о продажах и отвечать на команды в **вашем собственном** Telegram-боте, который вы создаёте самостоятельно. Настройка доступна на тарифах PRO и ULTIMATE.

## Настройка

1. Создайте бота через официального [@BotFather](https://t.me/BotFather) в Telegram (команда `/newbot`) и получите его токен.
2. Узнайте свой числовой Telegram chat ID (например, через бота вроде @userinfobot).
3. В панели откройте значок настроек (шестерёнку) в сайдбаре → **Notifications**.

**\[МЕСТО ДЛЯ СКРИНШОТА: окно Notifications]**

4. Заполните поля:
   * **Bot token** — токен вашего бота от BotFather.
   * **Chat ID** — ваш числовой chat ID.
   * **TG proxy** (необязательно) — прокси для бота в формате `ip:port:login:password`, если Telegram заблокирован в вашей сети.
5. Нажмите **SAVE**.

Панель начинает опрашивать вашего бота на новые сообщения и команды, реагируя только на сообщения от указанного chat ID — команды и сообщения от других пользователей полностью игнорируются.

Кнопка **UNLINK** отвязывает бота — уведомления и команды перестают работать до повторной настройки.

Если у аккаунта закончилась/приостановлена подписка (тариф ниже PRO) — бот сам временно перестаёт отвечать на команды, и снова заработает, как только подписка восстановится.

## Автоматические уведомления

Бот сам, без каких-либо команд, присылает уведомление при продаже предмета на market.csgo, Waxpeer или DMarket:

```
Продано
Аккаунт: <имя>
Площадка: <Market.CSGO / Waxpeer / DMarket>
Предмет: <название>
Сумма: <сумма> <валюта>
Время: <время>
```

Также бот пришлёт отдельное сообщение, если истёк токен лицензии панели или если у аккаунта подряд накопилось много сетевых ошибок прокси — это сигнал проверить прокси этого аккаунта.

## Команды

Отправьте боту `/help` (или `/h`), чтобы увидеть список команд прямо в чате.

### /bal

Показывает балансы по всем аккаунтам на всех площадках сразу (market.csgo, Steam, Waxpeer, DMarket), в их собственной валюте, включая замороженный остаток. Список постраничный, листается кнопками под сообщением, есть общий итог по каждой валюте вверху.

<figure><img src="../.gitbook/assets/image (64).png" alt=""><figcaption></figcaption></figure>

### /acc

Показывает, что конкретно сейчас выставлено на продажу, по каждому аккаунту — только те площадки, что включены тумблером для этого аккаунта. Одинаковые предметы группируются со средней ценой (например `x3 (средн. 12.50 USD)`). Список тоже постраничный.

<figure><img src="../.gitbook/assets/image (59).png" alt=""><figcaption></figcaption></figure>

### /transfer

Перевод баланса между аккаунтами на market.csgo или Waxpeer — полностью через кнопки, без ручного ввода сумм (переводится весь баланс целиком):

1. Выберите площадку — Market.CSGO или Waxpeer.
2. Выберите режим — **Между своими** (перевод с одного своего аккаунта на другой) или **Собрать со всех** (объединить балансы всех аккаунтов на одном).
3. Выберите аккаунт-источник (при переводе между своими).
4. Выберите получателя — из списка своих аккаунтов, либо нажмите **Другой получатель** и введите вручную API-ключ (market.csgo) или SteamID64 (Waxpeer) стороннего получателя.
5. Подтвердите операцию.
