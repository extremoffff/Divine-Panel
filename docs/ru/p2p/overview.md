# Обзор раздела Items

Раздел **P2P Marketplaces** (в сайдбаре) открывает главную таблицу панели — **Items** — где видны все предметы ваших аккаунтов, выставленные (или готовые к выставлению) на market.csgo и Waxpeer. Доступен на тарифах PRO и ULTIMATE.

<figure><img src="../.gitbook/assets/image (33).png" alt=""><figcaption></figcaption></figure>

## Верхняя панель

* **Trade status** — индикатор состояния активных трейдов/выдач предметов.
* **Profit** — открывает настройки калькулятора автоматических порогов, см. [Цена закупки и калькулятор прибыли](buy-price-profit.md).
* **Переключатель игры** (иконка CS2/значок игры) — market.csgo (со своими доменами по играм) и Waxpeer работают со всеми четырьмя играми одновременно в фоне; этот переключатель влияет только на то, что показано в таблице, а не на то, что реально репрайсится.
* **Показывать только для аккаунта** — фильтр таблицы по конкретному аккаунту (по умолчанию «All accounts» — показываются предметы всех аккаунтов сразу).
* Счётчик профилей/строк — сколько аккаунтов и сколько строк предметов сейчас в таблице.

## Вкладки и поиск

* **Tradeable** — предметы, готовые к продаже прямо сейчас (не в трейд-локе Steam).
* **Trade locked** — предметы, которые Steam ещё не разрешает передавать (обычный трейд-лок после получения/покупки предмета).
* Поле поиска — фильтр таблицы по названию предмета.

## Колонки таблицы

| Колонка         | Значение                                                                                                                        |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| Item / profiles | Название предмета и список аккаунтов, у которых он есть                                                                         |
| Qty             | Количество единиц этого предмета                                                                                                |
| Market min      | Минимальная цена этого предмета на площадке прямо сейчас                                                                        |
| Avg             | Средняя цена выставленных предметов                                                                                             |
| Buy price       | Ваша цена закупки предмета (если задана)                                                                                        |
| Profit          | Расчётная прибыль/убыток при продаже по текущей цене, с учётом комиссий площадки. Показывается, если указано значение Buy price |
| Threshold       | Заданный вами порог цены — минимальная (и, если задан, максимальная) цена, ниже/выше которой панель не будет выставлять предмет |

## Как панель решает, что и почём выставлять

Панель регулярно (каждые несколько секунд) пересчитывает актуальные цены по каждому предмету на каждой включённой площадке и либо выставляет предмет впервые, либо перевыставляет уже выставленный лот по новой цене — в рамках заданного вами порога (Threshold). Если для предмета не задан порог в валюте площадки — предмет просто не будет выставляться в этой валюте.
