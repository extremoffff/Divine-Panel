# Цена закупки и калькулятор прибыли

## Цена закупки (Buy price)

Чтобы колонка **Profit** в таблице показывала реальную прибыль, а не просто текущую цену, панели нужно знать, сколько предмет стоил вам изначально.

Кликните по ячейке цены закупки предмета — откроется окно **Set buy price** с полями для каждой валюты: **Buy price USD**, **Buy price RUB**, **Buy price EUR**.

<figure><img src="../.gitbook/assets/image (36).png" alt=""><figcaption></figcaption></figure>

Заданная цена закупки используется вместе с комиссиями конкретной площадки (продажа + вывод средств) для расчёта итоговой прибыли:

* MarketCSGO: 5% + 5%
* Waxpeer: 6% + 1%

## Автоматические пороги по проценту прибыли

Вместо того чтобы вручную выставлять порог для каждого предмета, можно один раз попросить панель рассчитать пороги автоматически — исходя из желаемого процента прибыли, для всех предметов, у которых уже задана цена закупки.

Нажмите кнопку **Profit** на верхней панели таблицы Items (или аналогичную на страницах DMarket/Steam Manager) → откроется окно **Automatic thresholds**.

<figure><img src="../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure>

1. Укажите **Target net profit/loss %** — целевой процент прибыли после вычета комиссий площадки. Например, `10` означает +10% прибыли; можно указать и отрицательное значение, например `-15`, если вы готовы продать даже с убытком до 15%.
2. Отметьте, для каких аккаунтов применить расчёт (кнопки **Select all** / **Select none** и список аккаунтов).
3. Нажмите **APPLY**.

Это разовое действие: панель один раз посчитает и сохранит новый порог Min для всех подходящих предметов (у которых задана цена закупки) выбранных аккаунтов — существующий порог будет перезаписан. Дальше пороги не пересчитываются автоматически при изменении цены закупки — при необходимости нужно запустить расчёт заново.

Важно: для автоматического просчета порогов у вас **обязательно** должны быть указаны цены покупки предметов (Buy price).

## Плавающий калькулятор прибыли

Отдельно от таблицы есть **плавающий калькулятор** — небольшое окно, которое можно перетаскивать по экрану и которое не закрывается при переключении между страницами панели. Подробнее — см. [Калькулятор прибыли](../tools/calculator.md).
