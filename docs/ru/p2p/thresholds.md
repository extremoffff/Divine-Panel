# Пороги цен (Threshold)

Порог — это цена, ниже которой панель никогда не выставит предмет на продажу. Без заданного порога предмет просто не будет автоматически выставляться в этой валюте.

## Как открыть окно порога

В таблице Items (или в аналогичных таблицах DMarket/Steam Manager Reprice) кликните по ячейке **Threshold** нужной строки предмета — откроется окно настройки порога для этого предмета.

**[МЕСТО ДЛЯ СКРИНШОТА: окно "Set threshold"]**

## Минимальный порог

Порог задаётся отдельно **для каждой валюты** — USD, RUB, EUR:

- **Min USD threshold**
- **Min RUB threshold**
- **Min EUR threshold**

Один и тот же порог по конкретной валюте действует **на все площадки, включённые для этого аккаунта одновременно** — отдельного порога на market.csgo и отдельного на Waxpeer для одного и того же предмета не существует, порог общий.

Если поле для какой-то валюты оставить пустым — предмет просто не будет выставляться в этой валюте (но может продолжать выставляться в других валютах, если для них порог задан).

Даже если ваш порог задан ниже — каждая площадка сама применяет свой собственный минимум и не даст выставить дешевле:

- market.csgo: 0.005$ / 0.005€ / 0.5₽
- Waxpeer: 0.005$

## Максимальный порог (опционально)

Флажок **Also cap this item with a max threshold** включает дополнительные поля верхней границы цены — **Max USD/RUB/EUR threshold**. Если задать максимум, панель не будет выставлять предмет дороже этой цены, даже если рыночная цена выше.

**[МЕСТО ДЛЯ СКРИНШОТА: развёрнутые поля максимального порога]**

## Очистка порогов

Кнопка **Clear thresholds** в этом же окне убирает все заданные пороги (минимальные и максимальные) для этого предмета сразу по всем валютам.

## Автоматическая установка порогов

Задавать порог вручную для каждого предмета не обязательно — если для предметов уже указана цена закупки (Buy price), панель может рассчитать пороги автоматически исходя из желаемой прибыли. См. [Цена закупки и калькулятор прибыли](buy-price-profit.md).
