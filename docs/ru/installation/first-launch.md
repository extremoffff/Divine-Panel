# Первый запуск

## Распаковка

Распакуйте архив `DivinePanel-Setup-win64.zip` в любую папку на компьютере (например, на рабочий стол или в `C:\DivinePanel`). Внутри окажутся файл **`Launcher.exe`** и несколько сопутствующих папок/файлов — трогать их не нужно, всё должно оставаться рядом друг с другом.

**[МЕСТО ДЛЯ СКРИНШОТА: содержимое распакованной папки]**

## Запуск

Запускайте панель **только через `Launcher.exe`** — не через файлы внутри вложенных папок. Именно Launcher при каждом запуске проверяет, нет ли новой версии панели, и при необходимости сам её подтягивает — самой панели для этого ничего вручную обновлять не нужно.

При первом запуске (и при каждом следующем) `Launcher.exe` покажет небольшое окно с анимированным логотипом и индикатором загрузки, пока проверяет обновления — обычно это занимает несколько секунд. Затем сама панель откроется автоматически в браузере.

## Значок в трее

После запуска панель продолжает работать в фоне — значок появляется в системном трее Windows (рядом с часами). Закрытие вкладки браузера панель не выключает — она продолжает работать в фоне, чтобы репрайсинг и уведомления не прерывались.

Чтобы снова открыть панель — кликните по значку в трее и выберите **Открыть панель**. Чтобы полностью закрыть программу — выберите **Выход** в том же меню трея.

**[МЕСТО ДЛЯ СКРИНШОТА: меню значка в трее]**

## Единственный запущенный экземпляр

Если попытаться запустить `Launcher.exe` ещё раз, когда панель уже работает — повторный запуск просто откроет в браузере уже работающую панель, не создавая второй процесс.

## Требования

Панель — приложение для Windows. Отдельно устанавливать .NET или что-либо ещё не требуется — всё нужное уже входит в состав скачанного архива.
