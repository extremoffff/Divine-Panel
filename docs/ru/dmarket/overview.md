# Обзор DMarket

Раздел **DMarket** в сайдбаре — отдельная площадка, устроенная иначе, чем market.csgo/Waxpeer: предметы для продажи нужно сначала физически депонировать со Steam на баланс DMarket. Доступен на тарифах PRO и ULTIMATE.

Прежде чем начать, аккаунт нужно подготовить для DMarket так же, как для остальных площадок — см. [Настройка аккаунта и площадок](../accounts/account-settings.md#подготовка-аккаунта-prepare).

Страница DMarket состоит из двух вкладок: **Inventory manager** и **Price**.

## Inventory manager — управление инвентарём

1. Выберите подготовленный для DMarket аккаунт и игру из выпадающих списков вверху страницы.
2. Список инвентаря показывает предметы с фильтром по статусу: **All**, **In Steam** (ещё не депонированы), **In inventory** (на балансе DMarket, не выставлены), **On sale** (уже выставлены на продажу).
3. Отметьте нужные предметы галочками (или используйте **Select all** / **Select none**).

Доступные действия:

* **Deposit selected** — депонировать выбранные предметы со Steam на DMarket. Это один общий трейд-запрос на всю выбранную партию, а не отдельная передача по каждому предмету — прогресс отображается по этапам одной операции, а не по количеству предметов.
* **Remove from sale** — снять предмет с продажи. Обязательный шаг перед выводом: DMarket не позволяет вывести предмет обратно, пока он выставлен на продажу.
* **Withdraw selected** — вывести выбранные предметы с DMarket обратно в Steam-инвентарь.

## Price — пороги и цены

Вторая вкладка зеркально повторяет таблицу Items из раздела P2P, но только для DMarket:

| Колонка    | Значение                                    |
| ---------- | ------------------------------------------- |
| Title      | Название предмета                           |
| Qty        | Количество                                  |
| Market min | Минимальная цена на DMarket сейчас          |
| Avg        | Средняя цена выставленных предметов         |
| Buy price  | Цена закупки (если задана)                  |
| Profit     | Расчётная прибыль с учётом комиссий DMarket |
| Threshold  | Заданный порог цены                         |

Настройка порогов и цены закупки происходит точно так же, как в разделе P2P — см. [Пороги цен](../p2p/thresholds.md) и [Цена закупки и калькулятор прибыли](../p2p/buy-price-profit.md). Фоновый цикл репрайсинга на DMarket тикает каждые примерно 20 секунд — кнопка **↻ Refresh** позволяет обновить таблицу вручную, не дожидаясь очередного тика.
