# Знакомство с Telegram-ботом

Все вопросы подписки — оплата, пополнение баланса, получение токена для панели, реферальная программа — решаются через отдельного Telegram-бота [@divinepanel\_bot](https://t.me/divinepanel_bot). Панель сама по себе к этому боту не подключается — это разные приложения: бот выдаёт токен, панель этим токеном пользуется.

## Открытие бота

Перейдите по ссылке [t.me/divinepanel\_bot](https://t.me/divinepanel_bot) и нажмите **Start** (или отправьте команду `/start`).

<figure><img src="../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

Откроется главное меню со следующими кнопками:

| Кнопка              | Назначение                                              |
| ------------------- | ------------------------------------------------------- |
| Информация о панели | Открывает сайт divinepanel.com                          |
| Кабинет             | Личный кабинет — баланс, тариф, дата окончания подписки |
| Пополнение          | Способы пополнения баланса                              |
| Инструкция          | Ссылка на данное руководство                            |
| Язык                | Переключение языка бота (русский/английский)            |
| Скачать             | Ссылка на скачивание панели (GitHub)                    |
| Реф. система        | Ваша персональная реферальная ссылка                    |
| Поддержка           | Прямая ссылка на связь с поддержкой                     |

Кнопка **Меню** внизу почти каждого экрана бота возвращает в это главное меню.
