# Знакомство с Telegram-ботом

Все вопросы подписки — оплата, пополнение баланса, получение токена для панели, реферальная программа — решаются через отдельного Telegram-бота [@divinepanel_bot](https://t.me/divinepanel_bot). Панель сама по себе к этому боту не подключается — это разные приложения: бот выдаёт токен, панель этим токеном пользуется.

## Открытие бота

Перейдите по ссылке [t.me/divinepanel_bot](https://t.me/divinepanel_bot) и нажмите **Start** (или отправьте команду `/start`).

**[МЕСТО ДЛЯ СКРИНШОТА: главный экран бота после /start]**

Откроется главное меню со следующими кнопками:

| Кнопка | Назначение |
|---|---|
| Информация о панели | Открывает сайт divinepanel.com |
| Кабинет | Личный кабинет — баланс, тариф, дата окончания подписки |
| Пополнение | Способы пополнения баланса |
| Инструкция | Раздел находится в разработке — пользуйтесь этим руководством |
| Язык | Переключение языка бота (русский/английский) |
| Скачать | Ссылка на скачивание панели (GitHub) |
| Реф. система | Ваша персональная реферальная ссылка |
| Поддержка | Прямая ссылка на связь с поддержкой |

Кнопка **Меню** внизу почти каждого экрана бота возвращает в это главное меню.

## Смена языка бота

Нажмите **Язык** в главном меню, затем выберите **RU** или **ENG**. Выбор сохраняется для вашего аккаунта — при следующем открытии бота язык не сбросится.
