# Пополнение баланса

В главном меню бота нажмите **Пополнение**. Откроется список доступных способов:

**[МЕСТО ДЛЯ СКРИНШОТА: список способов пополнения]**

- **USDT, USDC (EVM)** — сети BEP-20 (BSC), Arbitrum, Polygon. Все три сети используют один и тот же адрес — они совместимы между собой (все на базе EVM).
- **USDT (TON)**
- **USDT (Aptos)**
- **USDT, USDC (Solana)**
- **Bybit UID** — внутренний перевод внутри Bybit по UID (не по сети).
- **CryptoBot +3%** — оплата любым активом внутри Telegram-бота @CryptoBot; комиссию CryptoBot 3% нужно заплатить сверх суммы пополнения.

## Пополнение криптовалютой (EVM / TON / Aptos / Solana)

Выберите нужный способ — бот покажет ваш персональный адрес для пополнения:

**[МЕСТО ДЛЯ СКРИНШОТА: экран с адресом для пополнения, например EVM]**

1. Отправьте перевод на показанный адрес в одной из поддерживаемых сетей.
2. Дождитесь подтверждений сети — обычно баланс зачисляется автоматически в течение нескольких минут.
3. Страница отслеживает поступление депозита ограниченное время (обычно около 20 минут). Если перевод пришёл позже — просто откройте раздел пополнения этим же способом ещё раз, отслеживание включится заново.

Адрес закреплён персонально за вашим аккаунтом в боте и не меняется — можно пополнять его многократно.

## Пополнение через Bybit UID

1. Нажмите **Bybit UID** в списке способов пополнения.
2. Если Bybit UID ещё не привязан — нажмите **Указать Bybit UID** и пришлите его сообщением (число, которое видно в приложении Bybit: Профиль → UID).
3. После привязки бот покажет получающий UID Divine Panel — переведите USDT **внутренним переводом по UID** (не по сети/блокчейну).
4. Баланс зачислится автоматически в течение нескольких минут после перевода.

Привязанный UID можно поменять кнопкой **Изменить UID**.

## Пополнение через CryptoBot

1. Нажмите **CryptoBot +3%**.
2. Пришлите сообщением сумму в USD, которая должна зачислиться на баланс (например `10` или `10.5`). Сумма должна быть от 1 до 100000.
3. Бот создаст счёт (инвойс) в @CryptoBot — сумма к оплате будет немного больше указанной, так как сверху добавляется комиссия CryptoBot 3%.
4. Нажмите кнопку **Оплатить** — откроется CryptoBot, где можно расплатиться любым доступным активом (USDT, TON, BTC, ETH и другими) по внутреннему курсу CryptoBot.
5. После оплаты баланс зачисляется автоматически.

**[МЕСТО ДЛЯ СКРИНШОТА: экран создания счёта CryptoBot]**
