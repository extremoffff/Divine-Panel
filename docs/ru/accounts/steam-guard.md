# Steam Guard: коды и подтверждения

Каждая карточка аккаунта в Account Manager показывает код Steam Guard и счётчик активных подтверждений — работать с ними можно прямо из панели, не заходя в приложение Steam Guard на телефоне.

**[МЕСТО ДЛЯ СКРИНШОТА: карточка аккаунта с кодом Steam Guard и счётчиком подтверждений]**

## Код авторизатора

Пятизначный код обновляется автоматически каждые 30 секунд — под ним показан прогресс-бар текущего цикла. Код рассчитывается панелью самостоятельно на основе загруженного maFile, без обращения к серверам Steam.

Кликните по коду — он скопируется в буфер обмена, появится отметка «Скопировано».

Если код не отображается или показывает ошибку — проверьте, что у аккаунта загружен корректный файл maFile (поле при добавлении/редактировании аккаунта).

## Подтверждения (Confirmations)

Рядом с кодом — кнопка-счётчик, показывающая число активных подтверждений Steam (запросы на подтверждение трейдов, выставление предметов на продажу и так далее). Восклицательный знак вместо числа означает ошибку получения списка.

Нажмите на счётчик — откроется список активных подтверждений:

**[МЕСТО ДЛЯ СКРИНШОТА: попап со списком подтверждений]**

Для каждого подтверждения показаны иконка, заголовок и краткое описание; для трейдов дополнительно виден партнёр по обмену (SteamID64, кликабельный — копируется по клику) и превью предметов. У листингов есть отдельный бейдж «LISTING».

Каждую строку можно подтвердить (✓ Accept) или отклонить (✕ Decline) отдельно.

## Автоматическое принятие

Чтобы не подтверждать каждое действие вручную, в панели есть настройки автоматизации. Нажмите значок шестерёнки **Settings** на верхней панели страницы Account Manager — откроется панель настроек:

**[МЕСТО ДЛЯ СКРИНШОТА: выпадающая панель настроек Steam Guard]**

- **Auto-accept listings** — автоматически принимать подтверждения на выставление предметов на продажу.
- **Auto-accept gifts (empty trades)** — автоматически принимать входящие трейды, которые ничего не забирают с вашей стороны (подарки).
- **Poll interval** — как часто (в секундах, минимум 5) панель проверяет Steam на новые подтверждения и трейды.

Настройки применяются сразу при изменении и действуют для всех аккаунтов.
