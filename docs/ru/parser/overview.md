# Обзор Parser

Раздел **Parser** сравнивает цены и наличие предметов между двумя выбранными торговыми площадками одновременно — удобно для поиска возможностей купить дешевле на одной площадке и продать дороже на другой. Доступен только на тарифе ULTIMATE (либо по отдельному токену парсера, см. [Личный кабинет → Парсер](../subscription/cabinet.md#парсер-только-для-ultimate)).

## Выбор игры и площадок

Сверху страницы:

* Переключатель игры — **CS2, Dota 2, Rust, TF2**.
* **Platform 1** и **Platform 2** — два выпадающих списка с десятками поддерживаемых площадок (Steam, market.csgo, Waxpeer, DMarket, CSFloat, Skinport, Buff.Market, LIS-Skins и многие другие — набор конкретных площадок зависит от выбранной игры).
* Кнопка ⇄ между ними — быстро поменять платформы местами.
* Переключатель валюты (по умолчанию USD).
* Кнопка со значком валюты — курсы обмена: автоматические (живые) или заданные вручную.

## Фильтры

У каждой из двух платформ — своя панель фильтров, плюс отдельная панель общих фильтров:

**Фильтры по каждой платформе:**

* Цена от / до
* Количество (Qty) от
* Float от / до (используется как индикатор износа/качества, доступен не для всех площадок)
* Максимальное число дней бана (Ban days ≤)
* Переключатели StatTrak / Souvenir / Обычный / Только наклейки

**Общие фильтры:**

* Минимальный/максимальный процент прибыли (Profit %)
* Скрывать перенасыщенные позиции на площадке (Hide overstock)
* Минимальное число продаж за 7 дней — отдельно для Steam, MarketCSGO и DMarket

<figure><img src="../.gitbook/assets/image (49).png" alt=""><figcaption></figcaption></figure>

Все фильтры применяются сразу к таблице результатов, отдельной кнопки «Применить» нет.

## Автообновление

Флажок автообновления и поле интервала (в секундах, минимум 15) рядом с ним включают периодическое автоматическое обновление таблицы без ручного нажатия **Refresh**.

## Поиск и обновление

Поле поиска фильтрует таблицу по названию предмета. Кнопка **↻ Refresh** запрашивает свежие данные вручную.
